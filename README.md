This is my Project 1 submission for the DecodeLabs AI Internship, Batch 2026.
I built a rule-based AI chatbot in pure Python with no external libraries.
The bot uses a dictionary-based Hash Map for O(1) intent matching instead of an if-elif ladder.
It follows the IPO model — Input sanitization, Process logic, and Output via a continuous loop.
This project helped me understand the foundation of AI before moving into machine learning.
